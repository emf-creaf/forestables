# FFI example output

A subset of plots from the *Loire* department for 2015 to 2018.

## Usage

``` r
ffi_output_example
```

## Format

### `ffi_output_example`

A nested dataframe with the FFI plots for the selected years in the
*Loire* department

## Source

FFI raw data obtained from <https://inventaire-forestier.ign.fr/>
