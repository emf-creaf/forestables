# FIA example output

A subset of plots from Alaska (AK) for 2015 to 2018.

## Usage

``` r
fia_output_example
```

## Format

### `fia_output_example`

A nested dataframe with the FIA plots for the selected years in Alaska

## Source

FIA raw data obtained from
<https://www.fs.usda.gov/research/programs/nfi#data-and-tools>
