# forestables: Making Forest Inventory Data Tidy

The \`forestables\` package converts raw forest inventory files from the
Spanish Forest Inventory (IFN), the French Forest Inventory (FFI) and
the USA Forest Inventory (FIA) into tidy tables to be used in analyses
and modelling.
