# IFN example output

A subset of plots from *Barcelona* province for the IFN2, IFN3 and IFN4
versions.

## Usage

``` r
ifn_output_example
```

## Format

### `ifn_output_example`

A nested dataframe with the IFN plots for the selected versions in
*Barcelona* province

## Source

IFN raw data obtained from
<https://www.miteco.gob.es/es/biodiversidad/temas/inventarios-nacionales/inventario-forestal-nacional/cuarto_inventario.html>
